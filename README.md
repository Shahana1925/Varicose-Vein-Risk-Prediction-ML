# Varicose-Vein-Risk-Prediction-ML
Machine Learning based risk prediction model for varicose veins using Random Forest.


>>>> # Varicose Vein Risk Prediction using Machine Learning <<<<<<

## Overview
This project implements a Random Forest-based machine learning model to estimate the probability of varicose vein risk using clinical and lifestyle parameters.

## Features Used
- Age
- BMI
- Standing Hours
- Family History
- Pregnancy History
- Smoking

## Model
- Random Forest Classifier
- Train-test split validation
- Probability-based prediction output

## Tools
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Google Colab

## Output
The model predicts the probability of low and high risk and visualizes the results using a bar graph.
